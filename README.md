VincentBot – Aplikacja Badawcza

Projekt licencjacki realizowany na kierunku Psychologia i Informatyka (Uniwersytet SWPS). Aplikacja służy do przeprowadzania eksperymentu psychologicznego z wykorzystaniem sztucznej inteligencji.

Link do aplikacji: https://vincentbottest.streamlit.app/

O projekcie: VincentBot to chatbot oparty na koncepcji samowspółczucia (Self-Compassion). Wykorzystuje architekturę RAG (Retrieval-Augmented Generation), aby dostarczać wsparcie oparte na literaturze naukowej, minimalizując ryzyko błędów modelu.

Użyte technologie:
Python 3.11
Streamlit (Interfejs webowy)
LangChain (Orkiestracja modelu AI)
OpenAI GPT-4o mini (Silnik konwersacyjny)
FAISS (Wektorowa baza wiedzy)
Google Sheets API (Zapis wyników badania)
